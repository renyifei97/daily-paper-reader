## 摘要
自适应交通信号控制（ATSC）旨在通过实时调整交通信号灯来优化交通流并最大限度地减少延迟。多智能体强化学习（MARL）的最新进展在 ATSC 领域展现出巨大潜力，但现有方法仍受限于表征能力不足，在复杂多变的交通环境中往往导致次优性能和较差的泛化能力。另一方面，大语言模型（LLMs）在语义表征、推理和分析方面表现卓越，但其幻觉倾向和缓慢的推理速度往往阻碍了它们在决策任务中的直接应用。为了应对这些挑战，我们提出了一种名为 LATS 的新型学习范式，该范式整合了 LLMs 和 MARL，利用前者的强大先验知识和归纳能力来增强后者的决策过程。具体而言，我们引入了一个即插即用的师生学习模块，其中经过训练的嵌入式 LLM 作为教师，生成捕捉每个路口拓扑结构和交通动态的丰富语义特征。随后，一个更简单的（学生）神经网络通过潜在空间中的知识蒸馏来学习模拟这些特征，使最终模型能够独立于 LLM 运行，用于下游的强化学习决策过程。这种整合显著增强了整体模型在不同交通场景下的表征能力，从而实现了更高效且更具泛化性的控制策略。在多种交通数据集上的广泛实验表明，我们的方法增强了强化学习模型的表征学习能力，从而在整体性能和泛化性上均优于传统的强化学习和仅使用 LLM 的方法。

## Abstract
Adaptive Traffic Signal Control (ATSC) aims to optimize traffic flow and minimize delays by adjusting traffic lights in real time. Recent advances in Multi-agent Reinforcement Learning (MARL) have shown promise for ATSC, yet existing approaches still suffer from limited representational capacity, often leading to suboptimal performance and poor generalization in complex and dynamic traffic environments. On the other hand, Large Language Models (LLMs) excel at semantic representation, reasoning, and analysis, yet their propensity for hallucination and slow inference speeds often hinder their direct application to decision-making tasks. To address these challenges, we propose a novel learning paradigm named LATS that integrates LLMs and MARL, leveraging the former's strong prior knowledge and inductive abilities to enhance the latter's decision-making process. Specifically, we introduce a plug-and-play teacher-student learning module, where a trained embedding LLM serves as a teacher to generate rich semantic features that capture each intersection's topology structures and traffic dynamics. A much simpler (student) neural network then learns to emulate these features through knowledge distillation in the latent space, enabling the final model to operate independently from the LLM for downstream use in the RL decision-making process. This integration significantly enhances the overall model's representational capacity across diverse traffic scenarios, thus leading to more efficient and generalizable control strategies. Extensive experiments across diverse traffic datasets empirically demonstrate that our method enhances the representation learning capability of RL models, thereby leading to improved overall performance and generalization over both traditional RL and LLM-only approaches. *   **研究背景**：自适应交通信号控制（ATSC）是提升城市交通效率的关键。目前主流的**多智能体强化学习（MARL）**方法虽然具有实时性，但在复杂动态环境下表征能力有限，难以处理多样化的路口拓扑和流量变化。
*   **核心矛盾**：**大语言模型（LLM）**具备极强的语义理解和推理能力，但直接用于实时控制存在“幻觉”风险、推理速度极慢且计算成本高昂。
*   **核心问题**：如何既能利用 LLM 丰富的先验知识和语义表征能力，又能保持强化学习在实时决策中的高效性与鲁棒性？

### 2. 方法论：LATS 框架
LATS 提出了一种**教师-学生（Teacher-Student）学习范式**，将 LLM 的知识蒸馏到轻量级的 RL 模型中：
*   **核心思想**：在训练阶段，利用 LLM 作为“教师”生成路口拓扑和交通动态的语义嵌入；通过知识蒸馏，让“学生”网络（简单的神经网络）在潜在空间中模拟这些特征。在执行阶段，完全脱离 LLM，仅靠学生网络进行高效推理。
*   **关键技术细节**：
    1.  **提示词构建（Prompt Construction）**：将路口的静态拓扑（车道数、限速等）和动态状态（过去 4 个步长的车辆数、排队长度等）转化为结构化文本提示。
    2.  **教师模型（Teacher）**：使用预训练的嵌入式 LLM（如 `jina-embeddings-v2-small-en`）将提示词编码为高维语义向量。
    3.  **学生模型与潜在对齐（Latent Alignment）**：采用两个变分自编码器（VAE）。教师 VAE 编码语义特征，学生 VAE 编码原始状态向量。通过最小化两个潜在分布之间的 **KL 散度**，实现知识从 LLM 到 RL 模型的转移。
    4.  **RL 决策**：将学生网络提取的潜在语义特征与传统的交通状态特征融合，输入到基于 **PPO（近端策略优化）** 的决策头中，输出信号灯相位。

### 3. 实验设计
*   **数据集与场景**：
    *   **Grid 5×5**：包含 25 个相同路口的同质化网格网络。
    *   **Monaco**：包含 28 个具有不同拓扑结构的真实异质化摩纳哥路网。
*   **对比方法（Baselines）**：
    *   **传统方法**：Fixed-Time, Greedy, Max-Pressure。
    *   **纯 LLM 方法**：LLMLight（基于 Llama-2-7b 和 Llama-3-8B 直接决策）。
    *   **纯 RL 方法**：独立学习（MA2C, IQLD 等）和参数共享方法（IPPO, CoLight, AttendLight, HeteroLight）。
*   **评估指标**：排队长度、平均车速、路口延迟、行程完成率、行程时间及行程延迟。

*   **实验规模**：论文进行了广泛的实验，包括在两个路网上的性能对比、跨场景的 **Zero-shot（零样本）迁移实验**、以及针对不同组件的**消融实验**。

### 6. 主要结论与发现
*   **性能领先**：LATS 在所有关键指标上均优于传统 RL 和纯 LLM 方法。在摩纳哥异质路网中，其行程时间比最强基准降低了约 6.7%。
*   **泛化能力强**：在零样本迁移实验中（如在摩纳哥路网训练，在 Grid 路网测试），LATS 表现出显著优于其他 RL 方法的鲁棒性。
*   **表征增强**：LLM 提供的语义特征能帮助模型更好地识别路口间的细微差别，从而学习到更具针对性的控制策略。

### 7. 优点与亮点
*   **即插即用**：教师-学生模块可以集成到现有的多种通用 RL 框架中（如 AttendLight），具有很强的扩展性。
*   **兼顾效率与深度**：通过蒸馏技术，成功解决了 LLM 无法实时部署的痛点，实现了“训练时重、推理时轻”。
*   **多模态融合**：首次将 LLM 的文本语义理解能力引入到 RL 驱动的交通信号控制中，提升了模型的表征上限。

### 8. 不足与局限
*   **训练成本高**：对 LLM 嵌入的依赖导致训练时间增加了数倍，限制了在大规模路网上的快速迭代。
*   **提示词依赖**：目前仍依赖预定义的提示词模板，如果交通环境出现模板未涵盖的极端异常（如事故、极端天气），语义表征可能失效。
*   **缺乏智能体间协作语义**：目前主要增强的是单个路口的表征，尚未显式利用 LLM 来建模路口间的通信或协同语义。
